# blob sprites

Place sprite sheets for the **blob** pet species in this directory.

## Folder Structure

Organize by color variant:

```

  cream/
    idle.png
    happy.png
    sad.png
    sleep.png
  pink/

  yellow/
```

## Sprite Format

- Each sprite sheet is a horizontal strip of frames at 48×48px per frame.
- Sprites are rendered with `image-rendering: pixelated` for crisp scaling at all display sizes.
- Use transparent backgrounds (PNG with alpha).
